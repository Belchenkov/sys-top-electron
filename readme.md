# SysTop

> Create application on Electron.js for monitors your system CPU


## Getting Started

Clone this repository locally :

``` bash
git clone
```

Install dependencies with npm :

``` bash
npm install
```

Run for production :

``` bash
npm start
```

Run for develop (with nodemon) :

``` bash
npm run dev
```

Build release :

``` bash
npm run package-mac
```
``` bash
npm run package-win
```
``` bash
npm run package-linux
```
